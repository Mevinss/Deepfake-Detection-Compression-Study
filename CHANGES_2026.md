
## Технические исправления

### Исправление MediaPipe
- **Проблема:** MediaPipe 0.10.32 изменил API (нет `solutions`)
- **Решение:** Добавлен fallback на OpenCV Haar Cascade
- **Результат:** Детектор лиц работает стабильно

### Исправление параметров модели
- **Проблема:** `TypeError: build_model() got an unexpected keyword argument 'model_name'`
- **Решение:** Исправлен параметр на `name` в `app.py`
- **Результат:** Модель загружается успешно
