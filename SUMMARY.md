## 🚀 Как запустить

### Быстрый старт (без датасета)

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить веб-приложение
python app.py

# 3. Открыть браузер
# http://127.0.0.1:5000
```

Приложение работает с предобученными весами ImageNet даже без датасета!

### Полная установка (с обучением)

```bash
# 1. Настроить окружение
python setup.py

# 2. Подготовить датасет
python download_dataset.py
# Следуйте инструкциям для загрузки данных

# 3. Обучить модель
python src/train.py --config configs/config.yaml

# 4. Запустить приложение
python app.py
```

### Для быстрого тестирования:
